# CoinAPI-js

CoinAPI is a simple API to communicate with your coin's daemon via javascript.

## Installation

```bash
npm install
```
and edit the config.js file to match your node's RPC info

## Usage

*/getdifficulty
*/getconnectioncount
*/getblockcount
*/getblockhash/[index]
*/getblock/[hash]
*/getrawtransaction/[txid]/[decrypt]
*/getnetworkhashps
*/getmoneysupply
*/getdistribution
